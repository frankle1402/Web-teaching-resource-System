# 教学资源平台商业模式分析

> 讨论日期：2026-01-11
> 讨论人：李智高（天津天堰科技股份有限公司医学教育研究院执行院长）

---

## 一、核心战略定位

### 真正的商业本质

> **通过众筹模式让全国医学老师构建教学资源，为天堰的全线产品提供持续的内容供给**

这不是一个单纯的「教学资源平台」，而是：

> **一个医学教育内容众包引擎，通过 UGC 模式为天堰的全线产品（硬件、软件、第三方合作）提供持续的内容供给**

### 战略优先级

- **首要目标**：内容资产积累（长期价值）
- **次要目标**：短期现金流

## 三、To B 模式详解

### 模式描述

- **交付方式**：SaaS 云服务，不提供本地化部署
- **收费方式**：按年订阅，10-20万/年
- **计费依据**：教师人数或学生人数
- **核心权益**：
  - 教师无限制使用 AI 生成教学资源
  - 访问全网公开的教学资源库
  - 学校内部资源私有化（不对外公开）
  - 学生可访问校内资源 + 公开资源

### 价值主张

学校购买的不只是「AI 生成工具」，而是：

1. **即开即用的医学教学资源库**（众筹积累的公共池）
2. **本校专属内容空间**（草稿、未发布资源）
3. **教师使用无限额**
4. **学生访问权限**

### 关键设计原则

| 原则 | 说明 |
|------|------|
| To C 是内容入口 | 免费吸引教师创作，内容必须公开，成为公共池 |
| To B 是变现引擎 | 机构付费买「优质内容库+私有空间」 |
| 内容公开是底线 | 即使是 To B 用户，公开发布的内容也进入公共池 |
| 私有仅限内部 | 机构可有「未发布」的内部资源，不影响公共池增长 |

## 四、To C 模式详解

### 模式描述

#### 方案一：免费增值模式

- 教师注册认证后可免费使用
- 每月有一定额度限制
- 超出额度需付费
- **前提条件**：发布的资源全网公开可用，其他人可使用、拷贝

#### 方案二：学生付费阅读（限制场景）

- 精品资源可设置收费（如 0.1-1 元）
- 平台与教师分成（平台 < 25%）
- 让教师通过平台创业创收

### To C 价值主张

| 用户类型 | 权益 | 目的 |
|----------|------|------|
| 免费教师 | 每月 N 次生成，内容必须公开 | 内容积累 |
| 付费教师 | 更多额度，仍需公开 | 小额收入 + 内容积累 |
| 认证讲师 | 收益分成（社会学习者付费） | 激励高质量内容 |

## 五、学生付费的伦理边界

### 决策立场

采用「限制场景」策略，规避主要伦理风险。

### 允许的场景

- **社会学习者**：护士资格考试备考人员、在职进修人员
- **跨校学习**：A 校学生学习 B 校老师的资源
- **继续教育**：医院在职培训

### 禁止的场景

- 教师向**本校本班**学生收费
- 资源与**课程考核挂钩**的情况

### 技术实现建议

- 学生注册时填写学校+专业
- 系统自动屏蔽「教师-本校学生」的付费关系
- 付费内容标注「仅限校外/社会学习者」

## 六、版权与授权设计

### 版权归属

采用「**教师所有 + 平台授权**」模式。

### 用户协议核心条款（建议）

```
1. 您保留对原创内容的著作权
2. 您授予平台以下永久、不可撤销的权利：
   - 在平台及关联产品中展示、传播
   - 供其他用户学习、引用
   - 用于天堰科技的教育产品（硬件配套、在线课程等）
   - 授权给天堰的合作伙伴使用
3. 当内容产生商业收益时，您有权获得分成（具体比例见分成规则）
4. 您承诺内容不侵犯他人权利，如有纠纷由您承担责任
```

### 分成机制

| 收益来源 | 教师分成 | 平台分成 | 说明 |
|----------|----------|----------|------|
| 学生付费阅读 | 75% | 25% | 类似滴滴模式 |
| 内容授权给第三方 | 30-50% | 50-70% | 批量授权时按比例 |
| 硬件预装内容 | 一次性稿费 | - | 邀请优质作者 |

## 七、天堰的内容使用方式

确定的使用方向：

1. **在线平台课程**：作为天堰在线学习平台的配套内容
2. **第三方内容授权**：可打包授权给其他企业使用
3. **硬件引导访问**：硬件通过二维码等方式，引导用户使用小程序或浏览器访问资源

## 八、启动策略

### 总体路径

**先 To C 积累，再 To B 变现**

### 第一阶段：种子用户（0-1000 教师）

| 策略 | 具体动作 |
|------|----------|
| **内部资源** | 天堰现有客户关系中的医学教师，邀请内测 |
| **利益绑定** | 前 100 名种子用户，未来内容收益分成上浮 10% |
| **降低门槛** | 不限制生成次数，快速产出内容 |
| **内容质量** | 设置「精选」机制，优质内容获得流量倾斜 |

### 第二阶段：内容验证（1000 资源）

- 筛选出 100 个高质量资源
- 预装到天堰硬件产品试水
- 收集用户反馈，验证内容质量

### 第三阶段：商业化

- 开放 To B 订阅
- 开放学生付费（限制场景）
- 第三方内容授权

## 九、风险与应对

| 风险 | 应对策略 |
|------|----------|
| **内容质量参差不齐** | 建立审核机制 + 用户评价 + 精选推荐 |
| **教师创作动力不足** | 荣誉激励（排行榜）+ 物质激励（分成）+ 职业价值（作品集） |
| **版权纠纷** | 明确用户承诺 + 建立投诉机制 + 购买保险 |
| **竞品模仿** | 先发优势 + 内容壁垒 + 天堰渠道绑定 |

---

## 十、后续待办事项

1. **完善用户协议**：特别是版权授权和分成条款，建议法务审核
2. **设计教师激励体系**：积分、等级、排行榜、认证讲师
3. **规划内容质量标准**：什么样的资源可以被「精选」
4. **明确与天堰产品的对接方式**：硬件扫码跳转的技术方案
5. **试题库扩展**：同样的众筹模式可延伸到试题库等内容类型

## 附录：两种模式对比

| 维度 | To B 模式 | To C 模式 |
|------|-----------|-----------|
| **收入来源** | 机构订阅费 | 教师付费/学生付费/广告 |
| **内容归属** | 机构内部私有 | 全网公开共享 |
| **对天堰的价值** | 现金流，但内容被锁在机构内 | 现金流较弱，但内容沉淀在平台 |
| **规模化难度** | 销售驱动，扩张慢 | 产品驱动，可能爆发 |
